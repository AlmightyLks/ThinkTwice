# ThinkTwice

## Wieso?

Ich vergesse selber manchmal welche Seiten in Ordnung sind zur Meinungsbildung und Informationsbeschaffung.

## Wie es funktioniert

ThinkTwice warnt beim Surfen auf deutschen Nachrichtenseiten vor wenig vertrauenswürdigen Quellen. Auf einer Treffer-Seite wird oben ein farbiger Banner eingeblendet (Warnung / Eingeschränkt / Hinweis) mit einer kurzen Begründung, verlinkten Quellen und der Konzernzugehörigkeit. Über das Erweiterungs-Popup sieht man die Einstufung der aktuellen Seite. Auf der Einstellungsseite können alle Einträge durchsucht, bearbeitet, hinzugefügt oder zurückgesetzt werden.

## Installation

Die Erweiterung ist noch nicht im Chrome Web Store oder Firefox Add-on Store verfügbar. Die Installation erfolgt daher manuell aus einem lokalen Ordner.

**Chromium-basierte Browser (Chrome, Edge, Brave, Opera, Vivaldi):**

1. [Releases](https://github.com/AlmightyLks/ThinkTwice/releases) öffnen und die neueste `.zip`-Datei herunterladen oder das Repository klonen.
2. Im Browser `chrome://extensions` öffnen, oben rechts den Entwicklermodus aktivieren.
3. Auf »Entpackte Erweiterung laden« klicken und den Ordner auswählen.
4. Die Erweiterung sollte nun in der Symbolleiste erscheinen.
